Primeiro deve-se instalar o Cloud SDK e rodar o comando
https://dl.google.com/dl/cloudsdk/channels/rapid/GoogleCloudSDKInstaller.exe

# gcloud init

Para fazer o download do proxy 
https://dl.google.com/cloudsql/cloud_sql_proxy_x64.exe

Renomeie o arquivo para cloud_sql_proxy.exe.

Iniciar o proxy:
./cloud_sql_proxy -instances=docelar:southamerica-east1:docelar=tcp:5432
